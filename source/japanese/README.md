## 取り付け

[拡張機能をダウンロード](https://github.com/datenstrom/yellow-extensions/raw/master/zip/japanese.zip)そしてzipファイルを `system/extensions` フォルダーにコピーします。 Safariを使用している場合は、右クリックします。
